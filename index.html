¿Cuales son las Variables de Javascript?

Hay 5 tipos de variables : tipo texto(string), número, booleanas , objetos y funciones

  1. String

    Los cadena literales se pueden especificar usando comillas simples o dobles,
    que se tratan de manera idéntica, o usando el carácter de comilla invertida

    const nombre = "Juan";
    const apellido = "Pérez";
    const nombreCompleto = nombre + " " + apellido;
    console.log(nombreCompleto); // "Juan Pérez"

  2. Numero

    const num1 = 10;
    const num2 = 5;
    const suma = num1 + num2;
    console.log(suma)

  3. Booleanos

    Una variable es booleana si se le asigna como contenido true ó false.

    const num1 = 10;
    const num2 = 5;
    const esMayorQue = num1 > num2;
    console.log(esMayorQue);

  4. Objetos

    Un objeto es un entidad independiente con propiedades y tipos.

    var persona = {
    nombre: "Juan",
    apellido: "Pérez",
    edad: 30,
    console.log(persona.nombre); // Juan
    console.log(persona.apellido);// Pérez
